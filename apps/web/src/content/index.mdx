## Installation

```js
<script src="https://cdn.jsdelivr.net/npm/cursor-js@1.0.0/dist/index.min.js"></script>
```

```sh
npm install cursor-js # or @react/cursor-js
```

## Usage

Initialize the cursor with the following function. Passing config object is optional. Check out the [API Reference](#) for all config options.

```js
const config = {
  /* ... */
};
initCursor(config);
```

## React

Somewhere in the root of your project:

```jsx
import Cursor from "cursor-js";

// ...

function Layout() {
  return <Cursor />;
}
```
